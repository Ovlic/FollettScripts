FollettScripts
--------------

Important scripts:
* converter.py
  * Update patron data
* update_patrons.py
  * Update emails for patrons (6th+: add student email, 5th and below: add parent emails)
* get_from_nu.py
   * Turns patron data into a copy-pastable string
* homeroom_updater.py
   * Update patron's homeroom based on grade
* password_stuff/password_maker.py
   * Create a random 5 letter word + random 4 number password
* password_stuff/student_accounts
   * This folder is for duckyscript creation to automate logging in and setting up user accounts
